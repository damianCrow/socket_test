These instructions assume you have node and npm installed.

- git clone https://github.com/facebookincubator/create-react-app
- git clone https://github.com/damianCrow/socket_test
- Place the 'socket_test' directory into the 'create-react-app' directory.
- Run `cd socket_test`
- Run `npm install`
- Run `npm start`
- Run `cd server` then run `node index.js`
- Boom, you're done.
